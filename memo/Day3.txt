테이블(table)
테이블이란 여러 종류의 데이터를 보기 좋게 정리하여 보여주는 표입니다.
HTML에서는 <table> 태그를 이용하여 표 및 웹사이트의 틀을 작성할 수
있습니다.

<table>
   <tr> <-하나의 층
	<td>하나의 sell,하나의 방</td><td></td>
   </tr>
</table>

1. <table> 태그로 시작하고, 종료합니다.
2. <tr> 태그로 테이블의 층을 구성합니다.
3. <th> 태그로 각 열의 제목을 나타내고, 
    <td> 태그로 테이블의 셀을 나누어줍니다.

<table> 속성
border : 테이블 테두리를 설정합니다.
width : 테이블의 가로 폭을 설정합니다.
height : 테이블의 세로 폭을 설정합니다.
(width,height의 단위는 보통 px,%를 많이 사용합니다.)
픽셀(pixel)
픽셀이란 픽쳐(pitcure)와 엘리먼트(element)의 합성어 입니다.
컬러를 구성하는 최소단위입니다.
align : 테이블의 위치를 설정합니다.

<tr>, <th>, <td> 속성
width : 셀의 가로 폭을 설정합니다. (tr은 적용 X)
height : 셀의 세로 폭을 설정합니다.
align : 셀 안의 텍스트의 위치를 설정합니다.


colspan, rowspan 속성
colspan : 가로로 열을 합칩니다. colspan="합칠 열의 개수"
rowspan : 세로로 행을 합칩니다. rowspan="합칠 행의 개수"

colgroup
태그 뒤에 나오는 컬럼(th 또는 td)에 적용할 스타일을 해당 태그에서 미리
적용할 수 있게 합니다. 특히 각 칼럼의 길이를 설정하는데 가장 많이
사용됩니다.

caption
표에 캡션(제목)을 붙일 때 사용합니다. 기본 위치는 표의 위 가운데입니다.

하이퍼링크(HyperLink)
하이퍼링크란 다른 페이지나 사이트로 연결시키는 링크(문자 또는 이미지)
를 말합니다.

<a href="사이트 or 이동할 문서의 위치">링크에 사용될 문자 or 이미지</a>
예) 
<a href="HTML_Day3_04.html">클릭하면 프론트엔드 수강생 페이지가 보입니다.</a>

target 속성
<a>태그가 연결된 문서를 어디에서 열지 설정합니다.
_blank : 링크로 연결된 문서를 새탭에서 열어줍니다.
이름 : 이름으로 설정된 페이지에서 문서를 열어줍니다.

FTP (file transfer protocol)
파일 전송 프로토콜은 하나의 호스트에서 다른 호스트로 파일을 복사하기 위해
TCP/IP에 의해 제공되는 표준기능입니다.
TCP포트인 21번을 표준으로 사용하고 있습니다.

닷홈,파일질라 업로드시 주의 사항★
1. html 폴더에 저장
2. 홈페이지(메인페이지, 첫페이지)의 이름을 index.html로 저장해야 합니다.
(대소문자 주의, 무조건 소문자로 작성해야 합니다.)

vsc - 파일질라 연결

1.익스텐션에서 ftp-simple 설치
2.F1버튼을 클릭후 ftp-config 선택 후 닷홈 IP, PW 설정
3.F1버튼을 클릭후 ftp-remote ~~ 선택
